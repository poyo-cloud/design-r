# DESIGN-R 2020

`index.html` をブラウザで開くと使えます。

## できること

- DESIGN-R®2020 の各項目を選択して総点数を自動計算
- `D3-E6s6I3CG6n0p0:21点` のような表記を自動生成
- カルテ貼り付け用テキストをワンクリックでコピー
- 入力内容をブラウザの `localStorage` に保存
- Size / Pocket は cm 入力から点数を補助計算

## ファイル

- `index.html`: 画面本体
- `styles.css`: UIスタイル
- `script.js`: 採点ロジックとコピー処理
